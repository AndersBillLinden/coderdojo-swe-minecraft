# Uppgifter

## Självöppnande dörr

En dörr som öppnar sig när spelaren kommer i närheten. Dörren får sin power som är belägen diagonalt nedanför den. Maskinen kräver
förutom ett commandblock även en comparator och en repeater samt redstone dust (wire).
